# TOPC-i7-14650HX-6750GRE-Tahoe-OC
## 配置

硬件  | 型号
---  | :--
处理器 | I7 14650HX
显卡 | AMD 6750gre
硬盘  |	 致钛5000 1T
内存  |	16GB 威刚 3200MHz
无线 + 蓝牙 | AX210 无线网卡
网卡 | Realtek 8125


![配置]

## 注意：

* 1.先用安装EFI进行安装。

* 2.安装完后更换正式用的EFI，重启后用不定驱动打无线网卡及声卡补丁。

* 3.如没有无线，自行用occ工具关闭蓝牙及无线网卡驱动。


## BIOS设置：

*无需更改默认即可。
